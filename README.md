# Análise Exploratória de Dados - Supermercado

Projeto desenvolvido com Python e Jupyter Notebook com o objetivo de realizar uma análise exploratória de dados de vendas de um supermercado.

## Tecnologias utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Objetivos da análise

O projeto busca responder perguntas relacionadas a:

- Categorias mais vendidas;
- Sazonalidade de produtos;
- Perfil etário dos clientes;
- Distribuição das vendas;
- Impacto da Black Friday e Natal;
- Ticket médio por faixa etária;
- Impacto das promoções nas vendas.

## Arquivos do projeto

- `projeto analise exploratoria jupyter.ipynb`
- `vendas_supermercado_com_cliente.csv`
